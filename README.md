# Ling-Universe（泠宇宙）

> 这是一个长期维护的「技术叙事宇宙」：安全工程、成长记录、技术哲学与科幻文学的交汇处。

---

## 🌌 这是什么？

**Ling-Universe** 是一个长期更新的叙事档案库，融合了：

- 网络安全与工程实践（红队/静态分析/工具链）
- 个人成长轨迹与思维演化
- 技术哲学与制度/文明观察
- 科幻叙事与世界观构建
- 情感与人性的克制表达（部分内容会保持私密）

我给它的定位是：**一位技术者在时代与系统中成长的记录与想象**。

---

## 🧭 核心理念

Ling-Universe 关心的不是“爽点”，而是一些更底层的问题：

- 技术如何塑造文明？安全又如何保护文明？
- 变强的代价是什么？孤独从何而来？
- 当规则与权力发生冲突，个体该如何自处？
- 理性与情感能否在同一具身体里共存？
- 未来社会的新范式——协维主义该如何构建？

这里的故事与文字，既是叙事，也是思考工具。

---

## 🏷 世界观定位

这个宇宙围绕一个核心人物展开：

> **泠石（Ling-Shi）**  
> 红队工程师 / 故事叙事者 / 宇宙观察者

他代表一种交叉气质：

- 工程师的冷静、克制、系统观
- 情感与关系中的脆弱与修复
- 对自由、秩序、权力与伦理的持续追问

用于网页端阅读（主题、样式、首页、发布版内容）。

- `docs/_config.yml`：站点配置
- `docs/novels/*`：发布到网页端的小说版本（面向阅读）

## 📚 阅读指引（从哪里开始？）

如果你第一次来，推荐顺序：

1. `worldbuilding/timeline.md`（宇宙骨架）
2. `characters/LingShi.md`（主角与精神气质）
3. `tech-fiction/`（技术叙事章节）
4. `philosophy/`（思考与价值观）

本宇宙支持“非线性阅读”：  
你可以像浏览档案一样，从任何入口进入。

## 🔐 现实与虚构边界

Ling-Universe 是一个“混合叙事空间”：

- 有虚构的故事，也有现实经验的影子
- 有象征与隐喻，不等同于自传
- 有情感，但会经过克制处理

读者可以把它视作：  
**以文学方式书写的思想与人生模型**。
